## Requirements

* `docker compose` >= 2.0

## Quick start

* Get Docker in your system
* Clone this repository [chevereto/v4-docker](https://github.com/chevereto/v4-docker)

```sh
git clone https://github.com/chevereto/v4-docker.git
```

* From the repository folder run `make demo`
